# 项目概述 python1046

本项目是一款基于深度学习的智能运动推荐系统，采用前后端分离的架构设计，前端使用 Vue3，后端基于 Flask 框架，搭配 MySQL 数据库进行数据管理，并通过 Python 编程语言及深度学习技术实现核心功能。
